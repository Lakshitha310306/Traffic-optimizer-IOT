
# Smart Traffic Optimizer 🚦

## Description
A smart traffic management system that uses Genetic Algorithm to optimize traffic signal timings and reduce congestion.

## Features
- Vehicle and Pedestrian Sensors
- Traffic Simulation Engine
- Genetic Algorithm Optimization
- Pareto-based Multi-objective Optimization
- Dashboard UI Visualization

## Technologies Used
- Java
- Maven
- OOP Concepts

## How to Run
1. mvn clean install
2. Run Main.java

## Output
- Optimized traffic signals
- Reduced waiting time
- Simulation results in CSV
